# Case Format

Add one JSON file per case in this directory. Each file must follow the schema used in `missing-ai-model.json`.

Required top-level fields:
- `slug`
- `title`
- `difficulty`
- `estimatedTime`
- `briefing`
- `correctCulprit`
- `solutionExplanation`
- `suspects`
- `evidence`
- `criticalClues`
